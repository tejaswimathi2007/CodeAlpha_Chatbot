# 🤖 Rule-Based Chatbot (Python)

## 📌 Project Description
This is a simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs like greetings and common questions.

## 🚀 Features
- User interaction through console
- Predefined chatbot replies
- Uses functions and loops
- Exit command support

## 🛠️ Technologies Used
- Python
- Functions
- If-elif statements
- Loops

## ▶️ How to Run
Run:
python chatbot.py
